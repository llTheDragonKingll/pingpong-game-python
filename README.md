# 🏓 Ping-Pong Game (Python Turtle)

A two-player Ping-Pong game built using Python Turtle graphics, featuring real-time gameplay, paddle controls, and score tracking.

---

## 🚀 Features

* 🎮 Two-player gameplay
* 🏓 Paddle controls
* ⚡ Ball movement with collision physics
* 📊 Live score tracking

---

## 🛠️ Tech Stack

* Python
* Turtle Graphics
* Object-Oriented Programming

---

## 📂 Project Structure

```
pong-game-python/
│
├── main.py
├── paddle.py
├── ball.py
├── scoreboard.py
```

---

## ⚙️ How to Run

```
python main.py
```

---

## 🎮 Controls

* Player 1:

  * W → Up
  * S → Down

* Player 2:

  * ↑ → Up
  * ↓ → Down
